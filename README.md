## Setup

1) Clone the repo
2) npm init -y.
3) npm install.
4) Create .env file with PORT=3030.

## End points

### `api/v1/signup` POST



* You can Sign-up by sending post request using postman.





### `api/v1/signin`


* You can signin by sending post request using postman.
